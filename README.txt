AllBirds slideshow — Raspberry Pi Zero W (no browser needed)
============================================================
Files (all sit together, side by side):
  birdshow.py   <- the program you run
  frames/       <- the 117 slide images it shows

RUN IT (on the Pi, one time install then run):
  sudo apt install -y python3-pygame
  python3 birdshow.py

CONTROLS:
  RIGHT = next bird   LEFT = previous bird
  SPACE = pause       Q or ESC = quit
