# Native Bird Lobby Display

A Raspberry Pi 4 slideshow system for displaying native bird species 
found on a local golf course in Monterey County, CA.

Runs on a Raspberry Pi 4 connected to a standard PC monitor as a 
kiosk-style display for guests/lobby areas.

## Hardware
- Raspberry Pi 4
- Standard PC monitor (HDMI)

## What It Does
- Loops through bird species slides (PowerPoint / images)
- Designed to run automatically on boot
- No user interaction needed
